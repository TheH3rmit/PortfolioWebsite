2D Dungeon Generator

This project is a 2D dungeon generator created as a submission for the Algorithms and Complexity course. It is based on the Binary Space Partitioning (BSP) algorithm, which is used to recursively divide a defined space into smaller regions. The generated dungeon adheres to the following features:

  • Development Environment: Built in Godot Engine using GDScript.
  • Room Creation: Rooms are generated on a 2D grid, with the number of rooms always being a power of two.
  • Customization: Adjustable parameters such as room padding, tile size, and world size for flexible dungeon generation.
  • Debug Mode: Includes a debug visualization feature that highlights leaf nodes with green rectangles.
  • Path Creation: Implements corridors between rooms for accessibility.

Key Code Highlights:

  • The project uses a tilemap to render the dungeon and its corridors.
  • Recursive space partitioning ensures balanced room layouts.
  • Provides a function to clear and regenerate the dungeon dynamically.

This dungeon generator serves as an educational demonstration of the BSP algorithm and procedural content generation techniques in game development.
